- 依赖关系可能是强制或选择的，内部或外部的。
- 这四种依赖关系可以组合成强制性外部依赖关系 、强制性内部依赖关系、选择性外部依赖关系或选择性内部依赖关系。
- # 依赖关系
	- | 关系           | 举例                                     |
	  | -------------- | ---------------------------------------- |
	  | 强制性依赖关系 | 先开发出硬件，才能测试                   |
	  | 选择性依赖关系 | 软件可以在虚拟机上完成也可以在硬件上部署 |
	  | 外部依赖关系   | 供应商送样后才能制作样机                 |
	  | 内部依赖关系   | 结构团队完成装配后才能测试               |
